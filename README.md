# Syndicate.ai 🚀

**Syndicate.ai** is an advanced, omnichain AI trading ecosystem that operates as a hybrid decentralized/centralized fleet of autonomous agents. It tracks whale activity on Solana and analyzes market data on centralized exchanges using Google Gemini AI, executing trades with built-in risk-guard logic.

## 🧑‍💻 Architecture Overview

### Is it Online or Local?
**Both.** 
- **Production (Online):** The platform is architected as a split-stack application. The Vite+React frontend is designed to be pushed live to Vercel/Netlify. The FastAPI backend is configured to run on Render/Railway using dynamic port binding and Tortoise-ORM (PostgreSQL). Media assets are offloaded to AWS S3.
- **Development (Local):** You can run the entire swarm completely offline/locally on your own machine. It utilizes SQLite for the database and a local `/uploads` folder if AWS keys are not present.

### Is this a Trading Bot or Agent?
It is a **multi-agent trading system**:
1. **The Brain (AI Agent):** `strategy_engine.py` calls the Gemini API to analyze market news, technicals, and sentiment to formulate a confident buy/sell/hold signal.
2. **The Tracker (Whale Agent):** `solana_tracker.py` connects via WebSocket to the Solana blockchain, continuously tracking specific high-alpha wallets for copy-trade signals.
3. **The Risk Guard (Execution Agent):** `order_manager.py` enforces a strict 2% account equity rule and protects against flash crashes (Hallucination Circuit Breaker) before sending the final trade order to Binance (via CCXT) or Solana (via Jupiter API).

## 🛠️ Quick Start & Setup Guide

### 1. Requirements
Ensure you have the following installed:
* Python 3.10+
* Node.js (v18+) and npm
* A Google Gemini API Key
* A Solana RPC URL (Optional, but recommended for speed)

### 2. Backend Setup
1. Clone the repository and navigate to the backend directory:
   ```bash
   cd syndicate-ai
   ```
2. Install the necessary Python packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Create a `.env` file in the root `syndicate-ai` directory and add your keys:
   ```env
   GOOGLE_API_KEY=your_gemini_key
   TELEGRAM_TOKEN=your_telegram_bot_token (optional)
   SOLANA_PRIVATE_KEY=your_solana_private_key_array # e.g. [1,2,3...]
   SOLANA_API_KEY=your_helius_rpc_key
   AWS_ACCESS_KEY_ID=optional_s3_key
   AWS_SECRET_ACCESS_KEY=optional_s3_secret
   ```
4. Start the backend swarm:
   ```bash
   python main.py
   ```
   *The server will start on `http://localhost:8000`. This will automatically spin up the database, the AI polling loop, and the real-time Solana indexing.*

### 3. Frontend Setup
1. Open a new terminal and navigate to the frontend:
   ```bash
   cd syndicate-ai/frontend
   ```
2. Install Node dependencies:
   ```bash
   npm install
   ```
3. Set your environment variables via a `.env` file in the `frontend/` directory (if your backend is not running on localhost:8000):
   ```env
   VITE_API_URL=http://localhost:8000
   ```
4. Start the Midnight Glass Dashboard:
   ```bash
   npm run dev
   ```
   *The UI will launch, letting you connect your wallet via $0 gas EIP-712 signatures, configure AI parameters, and view real-time market action!*
